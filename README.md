# Kalos Aligners

Official website for **Kalos Aligners** (Kalos Dental Technologies Private Limited) — invisible clear teeth aligners designed by expert orthodontists.

Built with **React + Vite**.

## Features
- Multi-page site (Adults / Teens / Kids, Pricing, How it works, About, Blog, Legal pages)
- Lead-capture eligibility wizard → WhatsApp
- Interactive smile preview (camera + upload)
- Before/after slider, product gallery, auto-sliding reviews
- Free rule-based chat assistant
- Full per-page SEO + structured data (SEO/GEO ready)
- Mobile-first, responsive

## Getting started
```bash
npm install     # install dependencies
npm run dev     # start dev server (http://localhost:5173)
npm run build   # production build -> dist/
npm run preview # preview the production build
```
